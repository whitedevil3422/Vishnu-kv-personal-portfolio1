# Vishnu K V - Personal Portfolio

This is the personal portfolio website for **Vishnu K V**, a Mechanical Engineer and Self-Taught Developer.

## Overview
The website showcases Vishnu's skills in:
- **Mechanical Engineering**: CAD Design, SolidWorks, AutoCAD.
- **Software Development**: Web Development, Python, Automation.

## Tech Stack
- **HTML5**
- **CSS3** (Custom Properties, Flexbox, Grid)
- **JavaScript** (Vanilla)

## Features
- Responsive Design (Mobile-First)
- Dark Mode / Modern UI
- Interactive Animations
- Contact Form Layout

## Setup
To run this project locally:
1. Clone the repository.
2. Open `index.html` in your browser.
